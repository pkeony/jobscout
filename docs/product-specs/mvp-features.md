# MVP 기능 명세

## 4대 핵심 기능

### 1. JD 분석 ✅ (Day 3-4 완료)

**입력:** 채용공고 URL 또는 텍스트
**출력:** 스킬 추출(필수/우대), 직무 요약, 회사 정보, 주요 업무, 복리후생

수용 기준:
- [x] URL 입력 시 크롤링 → 분석 자동 진행
- [x] 텍스트 직접 입력 fallback
- [x] 크롤링 실패 시 친화적 안내
- [x] 스킬을 필수/우대/기타로 분류
- [x] SSE 스트리밍 실시간 표시
- [x] JSON 파싱 실패 시 원문 fallback

### 2. 프로필 매칭 (Day 5)

**입력:** 사용자 프로필 (스킬, 경력) + JD 분석 결과
**출력:** 매칭 점수, 강점, 갭 분석

수용 기준:
- [ ] 프로필 입력 폼 (스킬 태그, 경력 텍스트)
- [ ] 매칭 점수 게이지 (CountUp 애니메이션)
- [ ] 강점/갭 리스트
- [ ] localStorage 프로필 저장

### 3. 자소서 생성 (Day 6)

**입력:** JD 분석 결과 + 프로필
**출력:** 맞춤 자소서 초안 (마크다운)

수용 기준:
- [ ] 마크다운 렌더링 (react-markdown)
- [ ] 클립보드 복사 버튼
- [ ] SSE 스트리밍 타이핑 효과

### 4. 면접 예상질문 (Day 6)

**입력:** JD 분석 결과
**출력:** 예상 질문 + 모범 답변

수용 기준:
- [ ] 질문 카드 UI
- [ ] 답변 접기/펼치기
- [ ] 기술/인성 질문 구분
